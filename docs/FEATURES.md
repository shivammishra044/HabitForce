## üéÆ Gamification System

### Experience Points (XP)

**Earning XP**
- Base XP for habit completion
- Bonus XP for streaks
- Progressive XP scaling with levels
- Special event bonuses
- Challenge completion rewards

**XP Calculation**
```
Base XP = 10 points
Streak Bonus = (streak_days / 7) * 5 points
Level Multiplier = 1 + (level * 0.1)
Total XP = (Base XP + Streak Bonus) * Level Multiplier
```
